📝 Todo List App

A simple and functional Todo List Application built using Node.js, Express, and MongoDB.
This project allows users to create, update, and delete tasks easily.

🚀 Features
➕ Add new tasks
📝 Update existing tasks
❌ Delete single task
🗑️ Delete multiple selected tasks
✅ Checkbox-based selection
🛠️ Tech Stack
Backend: Node.js, Express.js
Database: MongoDB
ODM: Mongoose
Frontend: EJS (Embedded JavaScript Templates)
Styling: External CSS & Internal CSS
